# lein-jdk-javac

A Leiningen plugin to run javac

## Usage

Put `[phi4grv/lein-jdk-javac "0.1.0"]` into the `:plugins` vector of your `:user`
profile or `project.clj`.

```
$ lein jdk-javac -Xlint javasrc/package/Example.java
```
